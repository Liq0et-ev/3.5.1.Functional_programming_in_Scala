# 3.5.1.Functional_programming_in_Scala
# Pirmreizinātāju sadalīšana 

Programma nolasa vienu veselu skaitli no ievades.<br>
<br>
Ja ievadītais skaitlis ir mazāks par 2, programma izvada paziņojumu "error".<br>
<br>
Ja skaitlis ir vienāds ar 2 vai lielāks, programma sadala to pirmreizinātājos un izvada pirmreizinātājus ekrānā.<br>
<br>
Izvadē tiek parādīti visi atrastie pirmreizinātāji (ar atkārtojumiem, ja tādi ir).
